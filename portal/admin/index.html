Admin Console
⚠ Admin identity management is temporarily disabled while the secure platform is being rebuilt.

Manage students & teachers in the Firebase Console for now

Creating, editing, disabling, assigning or deleting student/teacher records from a web page is turned off under the secure rules model (Phase 3D/4A). There is no admin secret in this page and anonymous writes are not enabled.

Use the Firebase Console → Firestore to manage the students and teachers collections until a dedicated admin-auth Cloud Function is built.

What a future secure Admin Console will do

  • Create teacher codes and student codes (server-side, via an admin-only Cloud Function).
  • Assign students to teachers and classes.
  • Disable / reactivate codes without deleting historical achievements.
  • All behind a real admin sign-in — never a shared secret in the browser.

All write actions on this page are intentionally disabled.
