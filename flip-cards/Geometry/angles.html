Teacher notes

Purpose

This version builds fluency with recognising angle relationships. Use the checkboxes to choose which angle types appear. Turn on Unknown angle to show x on the front, then reveal x = ...° plus the matching reason on the reverse.

Statements
  • Complementary angles add to 90°
  • Supplementary angles add to 180°
  • Vertically opposite angles are equal
  • Angles at a point add to 360°
Suggested routine

Ask students to identify the relationship before flipping. If unknown angle mode is on, ask them to first name the reason, then state the value of x and justify it.
